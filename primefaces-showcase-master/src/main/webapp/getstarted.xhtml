Getting Started PrimeFaces is a lightweight library with one jar, zero-configuration and no required dependencies. You just need to download PrimeFaces, add the primefaces-{version}.jar to your classpath and import the namespace to get started.

Download

PrimeFaces can be downloaded manually or via maven.

<dependency>
    <groupId>org.primefaces</groupId>
    <artifactId>primefaces</artifactId>
    <version>7.0</version>
</dependency>

Namespaces

PrimeFaces namespace is necessary to add PrimeFaces components to your pages.

xmlns:p="http://primefaces.org/ui"
